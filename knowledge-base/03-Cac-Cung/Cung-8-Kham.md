# Cung 8 — Khảm (Thủy Môn)

## Thông tin cơ bản

| Thuộc tính | Giá trị |
|---|---|
| Số cung | **8** |
| Phương vị | Chính Bắc |
| Quái | Khảm (坎) |
| Thuộc tính | **Dịch Khí** (易气) — dễ biến đổi (Âm) |
| Bát Môn | Hưu Môn (休门) — chủ về nghỉ ngơi, an cư |
| Chủ về vùng | Diễn Châu |

## Ý nghĩa

**Dịch Khí Âm** — biến đổi từ phía Âm. Khảm là Thủy, linh hoạt, chảy tràn.

**Hưu Môn** — nghỉ ngơi, an cư. Cửa tương đối cát (nghỉ không phải hung).
