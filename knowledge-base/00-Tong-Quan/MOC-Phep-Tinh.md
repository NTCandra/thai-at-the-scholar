# MOC — Phép Tính Thái Ất

## Chuỗi tính toán

```
Năm CN
  → Tích Niên (10,153,917 + năm)
    → Tích Niên % 72 = Số Cục (vị trí Thái Ất)
      → Thiên Mục (vị trí Văn Xương)
        → Khách Mục (vị trí Thủy Kích)
          → Chủ Toán / Khách Toán / Định Toán
            → Phân Loại Số → Luận Đoán
```
