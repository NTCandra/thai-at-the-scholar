# Tích Niên (积年)

> [!summary]
> Tích Niên là "số năm tích lũy từ Thượng Nguyên Giáp Tý đến năm cần tính" — nền tảng của tất cả phép tính Thái Ất. Mọi vị trí đều được suy ra từ con số này.

## Định nghĩa

**Tích Niên** = tổng số năm đã trôi qua kể từ điểm khởi đầu vũ trụ (Thượng Nguyên Hỗn Độn Giáp Tý) đến năm cần tính.

Theo *Thái Ất Kim Kính Thức Kinh*: Từ Thượng Nguyên đến năm Khai Nguyên thứ 12 (724 CN) = **1,937,281 năm**.

## Công thức

### Niên Kế và Nguyệt Kế

```
Tích Niên = 10,153,917 + Năm Dương Lịch
```

**Ví dụ — Năm 2004:**
```
Tích Niên = 10,153,917 + 2004 = 10,155,921
```

### Nhật Kế và Thời Kế

```
Tích Nhật = 29,277 + (số ngày từ điểm mốc)
```

Hằng số 29,277 là Tích Nhật của năm Giáp Tý.

## Từ Tích Niên suy ra

```
Tích Niên % 360 → vị trí trong đại chu kỳ
Tích Niên % 72  → Số Cục (vị trí Thái Ất)
Tích Niên % 60  → Can Chi năm
Tích Niên × 18 % 18 → cơ sở tính Thiên Mục
```

## Ví dụ đầy đủ (Năm 2004)

```
Tích Niên = 10,153,917 + 2004 = 10,155,921

→ Số Cục: 10,155,921 % 72 = 33  (Dương Cục 33)
→ Can Chi: 10,155,921 % 60 = 21  → Can Chi năm
→ Số quẻ:  10,155,921 % 64 = 17  → Quẻ Tùy (随)
```
